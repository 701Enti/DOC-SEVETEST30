esp_err_t PsP2P_DM_buy ( PsP2P_DM_node_handle_t  consumer,
const char *  producer_name,
const char *  product_name 
)

PsP2P_DM购买产品

参数
consumer消费者句柄
producer_name生产者名
product_name产品名
注解
产品名一般是产品数据资源source存储对象名,比如把结构体变量的指针作为产品数据资源source上架,那么产品名一般是结构体变量的变量名
消费者可以购买产品名相同,但是生产者名不同的商品,同一生产者不可以有完全同名产品,所以消费者不可能买到它们
返回
[ESP_ERR_INVALID_ARG 输入了无法处理的空指针]
[ESP_ERR_NOT_SUPPORTED 非消费者不可使用购买(buy)操作]
[ESP_ERR_INVALID_STATE 产品被过多消费者购买 / 节点链表为空链表,维护状态异常 / 该产品消费者节点列表存在未成功删除的消费者节点引用,维护状态异常 / 该生产者的产品链表存在头或尾缺失,维护状态异常]
[ESP_ERR_NOT_FOUND 未找到指定的生产者/找到生产者,但未找到指定的产品]
[ESP_ERR_NO_MEM 内存不足]

创建PsP2P_DM节点

参数

identity节点身份,这是一个枚举类型

返回
PsP2P_DM节点句柄 / [NULL 为空指针的节点名/内存不足/不允许创建同名同身份节点/节点链表存在头或尾缺失,维护状态异常]

销毁PsP2P_DM节点

注解

PsP2P_DM产品上架

参数
producer生产者句柄
resource产品的数据资源
product_name产品名
注解
产品名一般是产品数据资源source存储对象名,比如把结构体变量的指针作为产品数据资源source上架,那么产品名一般是结构体变量的变量名
同一生产者不可以有完全同名产品,上架时会检查产品名,若产品名被使用过(根据内容比较)不允许上架
返回
[ESP_OK 上架成功]
[ESP_ERR_NO_MEM 内存不足]
[ESP_ERR_INVALID_ARG 输入了无法处理的空指针 / 同一生产者不可以有完全同名产品,对本生产者,产品名在自己的产品链表中已经被使用过]
[ESP_ERR_INVALID_STATE 产品链表存在头或尾缺失,维护状态异常]
[ESP_ERR_NOT_SUPPORTED 非生产者不可使用上架(put)操作]
