◆ direct_draw()

void direct_draw ( int32_t  x,
int32_t  y,
const uint8_t *  p,
uint8_t  change 
)

彩色图像直显方式 取模方式适配Img2Lcd

[水平扫描，从左到右，从顶到底扫描，24位真彩（RGB顺序），需要图像数据头]

将自动获取图像头参数,之后ledarray_set_and_write需要被调用才可显示

参数
x图案横坐标(无范围限制，超出不显示)，灯板左上角设为原点（1，1），由左到右绘制
y图案纵坐标(无范围限制，超出不显示)，灯板左上角设为原点（1，1），由上到下绘制
p导入图像指针
change亮度调整（1-100）警告:过高的调整幅度可能导致色彩失真
